NEO SERVER

NEO SERVER ist ein auf der CCU3 vorinstalliertes Server-Programm, das in Kombination mit der leistungsstarken App-Visualisierungs-Software AIO CREATOR NEO arbeitet, deren Lizenz im Leistungsumfang der CCU3 enthalten ist.


AIO CREATOR NEO

Mit dem AIO CREATOR NEO gestalten Sie sich im Handumdrehen eine attraktive und vollständig individuelle Steuerungs-App für Ihre CCU3 und die daran angelernten Komponenten. Außerdem können Sie über das einzigartig flexible NEO-System viele weitere Smart Home Geräte anderer Marken/ Hersteller wie z.B. Audio-/ Video-Systeme, Netzwerkkameras uvm. problemlos mit Ihren HomematicIP-Geräten kombinieren und mit nur einer einzigen App steuern.



NEO SERVER bietet die Möglichkeit zur Erweiterung des NEO-Systems um Sprachsteuerung oder plattfor- mübergreifende Automatisierungen via mediola Cloud-Services oder über den NEO Automation Manager. Mit NEO Server wird die CCU3 zu einem 24/7-Server der zur Überwachung und Ausführung verschiedenster Aufgaben innerhalb des NEO-Systems dient.


NEO AUTOMATION MANAGER

Der NEO Automation Manager ist ein optionales NEO Plugin, mit dem es möglich ist, Geräte und Systeme aller in NEO integrierten Marken plattformübergreifend zu automatisieren. So werden alle Ihre Geräte auch wirklich „smart“, können selbständig auf Ereignisse reagieren und Ihnen Arbeit abnehmen.

Dabei ist die Realisierung der Automationen extrem intuitiv über ein sehr einfach bedienbares grafisches Frontend – den Blockeditor. Programmierkenntnisse benötigen Sie dafür keine. Alternativ können komple- xe Automatisierungen auch mittels Javascript programmiert werden.

Probieren Sie es einfach aus, Bis zu 2 Automatisierungsprogramme (Tasks) können Sie auch ohne Lizenz erstellen.


MEDIOLA CLOUD SERVICES

Die ebenfalls optionalen Mediola Cloud Services dienen u.a. dazu Ihre in das NEO-System integrierten Geräte mittels Amazon Alexa oder Google Assistant bequem per Sprachbefehl zu steuern. Auf diese Weise wird Ihr Smart Home noch smarter und komfortabler. So können Sie bspw. Licht, Rollläden oder den Fernseher aus dem Bett oder vom Lieblingssessel aus mittels Zuruf zu dirigieren. Einfach genial – genial einfach!
Außerdem lassen sich darüber NEO-Geräte ganz einfach auch in die umfangreiche Automatisierungslogik von IFTTT und Conrad Connect einbinden, um auf diese Weise bspw. geobasierte Auslösefunktionen (Geofencing) zu realisieren oder eine Verbindung mit Google Kalender herzustellen.
